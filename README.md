# JAVA Practice Repository

Welcome to this Java learning repository! It contains a collection of beginner-friendly Java programs, core concepts, and Data Structures & Algorithms (DSA) practice files.

## What’s inside

- Basic Java programs covering variables, operators, loops, and conditions
- Simple practice files for input/output and logic building
- DSA exercises such as searching, arrays, and trees
- A growing folder for problem-solving practice

## Project Structure

- [Average.java](Average.java) - basic Java practice
- [SUM.java](SUM.java) - simple arithmetic example
- [J3.java](J3.java) - Java practice file
- [JAVA1.java](JAVA1.java) - Java basics
- [while_loop.java](while_loop.java) - loop practice
- [DSA(30)](DSA(30)) - DSA and algorithm practice programs

## How to run a Java file

Open a terminal in this folder and run:

```bash
javac FileName.java
java FileName
```

Replace FileName with the Java file you want to execute.
